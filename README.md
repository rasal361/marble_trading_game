# 🎲 Marble Betting Game (Python)

A simple Python game where you **bet money and draw random marbles** to win or lose.

---

## 🚀 Features

* Random marble draw using Python
* Betting system
* Multiple rounds gameplay
* Win or lose money based on marble color

---

## 🧠 How It Works

* You start with **1000 gold**
* Play for **10 rounds**
* Each round:

  * Place a bet
  * A marble is drawn randomly
  * Your money changes based on the result:

| Marble | Result  |
| ------ | ------- |
| Green  | +Bet    |
| Red    | -Bet    |
| Black  | +10×Bet |
| White  | -5×Bet  |

* Game ends early if you lose **half your money**
